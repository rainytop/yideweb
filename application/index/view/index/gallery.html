产品画廊

{昶优益生菌} // {谓舒益生菌} // {芷敏舒益生菌} // {小分子人参活性肽}

  • 昶优益生菌益德成功研制出单位含量高达60亿株/支的益生菌。它活性极强，耐酸、耐氧、耐温并能自我繁殖，被誉为“超级益生菌”。后在多家机构的鼎力支持下，将双歧杆菌、保加利亚乳杆菌、嗜乳酸杆菌等多种益生菌进行科学配伍，形成肠道特效超级益生菌生物制剂。

  • 谓舒益生菌通过胃部益生菌的定植进行保护和修复胃粘膜，来达到治疗包括胃炎在内的各种胃部的目的。

  • 芷敏舒过敏是一种免疫疾病，是人体内免疫功用失调呈现不平衡的情况。过敏体质患者，在临床上调查的时分，往往发现多种过敏反响性疾病常可见于同一患者，与正常人比较，过敏体质患者血清IgE显着升高，肥大细胞数较多并且细胞膜上的IgE受体也较多。芷敏舒富含降低IgE抗体需补充获得降低血清IgE抗体的抗过敏菌株。
